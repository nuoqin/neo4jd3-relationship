## neo4jd3-relationship

### 简介
本系统主要用于关系图谱展示。可以用于neo4j图数据库展示，也可以自定义数据集进行展示。

由于本人不是专业的前端开发者，结合自己的理解和网上搜集，修改neo4jd3的部分代码，实现了以下的效果。
